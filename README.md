## Installation:
All builds and downloads are available [here](https://github.com/owstat/owstat/releases) in the repo releases section.

## Development

Installation:
```
$ git clone https://github.com/owstat/owstat.git
$ cd owstat
```

Running:
```
$ npm start
```

Packaging:
```
$ npm run package
```
